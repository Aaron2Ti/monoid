Mono16 Normal, SemiCondensed & Condensed
======
Monospaced fonts made for programming

· Download, issues & contribute on Github
· Share on Twitter or Facebook

## About
I couldn't find a crisp looking font for low-res displays at my prefered font size so I made these

· Semi-narrow        For those long lines of code
· Bold punctuation   .,:;·!?
· Big operators      +-*=<>%
· Distinguishable    1Il 0Oo ~-_ ([{
· Squeeze            AÅÄÃA EÉËÊE
· Asterisk           *
· Options            See readme.pdf

## Testdrive

Mono16 Normal

require "gem"

string = "base16"
symbol = :base16
fixnum = 0
float  = 0.00
array  = Array.new
array  = ['chris', 85]
hash   = {"test" => "test"}
regexp = /[abc]/

# This is a comment
class Person

  attr_accessor :name

  def initialize(attributes = {})
    @name = attributes[:name]
  end

  def self.greet
    "hello"
  end
end

person1 = Person.new(:name => "Chris")
print Person::greet, " ", person1.name, "\n"
puts "another #{Person::greet} #{person1.name}"
