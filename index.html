Mono16
======
A monospaced font made for programming

· Download
· Share on Twitter or Facebook
· Ideas, errors & requests on Github

## About
I couldn't find a font that looked crisp on my non-hdpi display at my prefered font size.
Some design choises I made:

· Semi-narrow to fit long lines of code
· Generous letter-spacing
· Bold punctuation .,:;·!?
· Big operators    +-*=<>%
· Distinguishable  1Il 0Oo ~-_ ([{
· Asterisk         * (can be changed)

## Roadmap
More character options (e.g. l, 1, 0) + other ideas?

  attr_accessor :name

  def initialize(attributes = {})
    @name = attributes[:name]
  end

  def self.greet
    "hello"
  end
end

person1 = Person.new(:name => "Chris")
print Person::greet, " ", person1.name, "\n"
puts "another #{Person::greet} #{person1.name}"
