Mono16
======
A monospaced font made for programming at 16px

· Download
· Share on Twitter or Facebook
· Suggest etc. on Github

## About
I couldn't find a font that looked crisp on my non-retina display at my prefered font size.
Some design choises I made:

· Semi-narrow to fit long lines of code
· Generous letter-spacing
· Bold punctuation .,:;·!?
· Big operators    +-*=<>%
· Distinguishable  1Il 0Oo -_ ([{
· Asterisk         *

## Roadmap
Character options (e.g. asterisk, l, 1, 0) + support most latin languages + other ideas?
